== Description ==

A handy plugin for hide contents and images from guest users. But you can control how many characters and what words show.


= Features =
* Supporting Post Types
* Change dummy image in content
* Visible some character to show
* Exclude phars to show

= What is next =
* Login/register form on hide content
* Settings page for form
* Settings for hidden span
* Settings for unhide image, audio, video, text, link and more
* Hook on comments
